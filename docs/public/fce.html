Course Description

The Fundamentals of Computer Engineering course covers the essential principles needed to understand computational logic, programming, and the design of computer systems. It is crucial for any student aiming to advance in the field of computer science.

Main Topics:

  • Programming Logic
  • Algorithms and Data Structures
  • Operating Systems
  • Computer Networks
  • Computer Architecture

Units Overview:

  1. Unit 1: Introduction to the background and history of computers.
  2. Unit 2: Understanding the current state and impact of computer science.
  3. Unit 3: Exploring computer engineering in the professional context.
  4. Unit 4: Learning about computer hardware and its components.
  5. Unit 5: Exploring the fundamentals of computer software.
  6. Unit 6: Introduction to operating systems and their functions.
  7. Unit 7: Basics of computer networks and connectivity.
  8. Unit 8: Analyzing new trends and innovations in technology.
  9. Unit 9: Understanding information management and data principles.
  10. Unit 10: Introduction to Big Data and its applications.
  11. Units 11 & 12: Introduction to the basics of software development.
  12. Unit 13: Learning the foundations of web programming.
  13. Unit 14: Introduction to HTML and its uses in web design.
  14. Unit 15: Basics of CSS and styling for web pages.
